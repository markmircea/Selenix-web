Version 2.4.1

December 15, 2024
Major Release

New Features

  • AI-Powered Element Detection - Automatically find the best selectors using machine learning
  • Visual Workflow Builder - New drag-and-drop interface for creating complex automations
  • Cloud Sync - Sync your workflows across multiple devices seamlessly
  • Advanced Scheduling - Set up complex schedules with multiple triggers and conditions
  • Error Recovery System - Automatic retry and fallback mechanisms for failed actions

Improvements

  • 50% faster execution speed for large workflows
  • Improved memory usage and resource optimization
  • Enhanced user interface with better accessibility
  • More reliable element detection in dynamic websites
  • Better error messages and debugging information

Bug Fixes

  • Fixed issue with browser extension not connecting on macOS
  • Resolved memory leaks in long-running workflows
  • Fixed data export formatting issues with special characters
  • Corrected timing issues with very fast websites
  • Fixed crash when handling very large datasets
