Installing Selenix

Installing Selenix is quick and easy. Follow the instructions for your operating system below to get started with browser automation.

System Requirements:

Selenix requires Chrome or Edge browser (version 88 or later) to be installed on your system.

Method 1: Download Installer (Recommended)

1

Download the installer

Get the latest version from our download page.

Download for Windows
2

Run the installer

Double-click the downloaded .exe file and follow the installation wizard.

3

Launch Selenix

Once installation is complete, you can launch Selenix from your Start menu or desktop shortcut.

Note for Windows Users:

If you encounter a Windows Defender SmartScreen warning, click "More info" and then "Run anyway". This happens because our installer is not yet widely distributed.

1

Open and Install

wget https://releases.selenix.io/selenix_latest_amd64.deb
sudo apt install ./selenix_latest_amd64.deb

Verification

To verify that Selenix is installed correctly:

Launch the desktop app

Open Selenix from your applications menu

Check browser extension

Ensure the Selenix icon appears in your browser toolbar

Test connection

Click the extension icon and verify it connects to the desktop app

Success!

If you see "Connected to Selenix" in the extension popup, you're ready to start automating!
