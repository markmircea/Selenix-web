Installation

Getting started with Selenix is quick and easy. Follow the steps below to install the standalone application on your system and start automating your browser tasks.

Installation

Selenix is a standalone browser automation application that works on Windows, macOS, and Linux. The installation process is simple and takes just a few minutes.

1

Visit the Chrome Web Store

Navigate to the Chrome Web Store by clicking the link or by searching for "Chrome Web Store" in your favorite search engine.

2

Search for Selenix

In the Chrome Web Store, use the search function to look for "Selenix" or use our direct link: Selenix Chrome Extension.

3

Add to Chrome

Click the "Add to Chrome" button and confirm the installation when prompted. The extension will be added to your Chrome browser automatically.

4

Pin the Extension (Optional)

For easy access, you can pin the Selenix extension to your Chrome toolbar. Click the extensions icon in the top right corner of Chrome, find Selenix, and click the pin icon.

5

Create an Account

Click on the Selenix icon in your toolbar and follow the prompts to create a new account or sign in to an existing one.

Chrome Extension

The Selenix Chrome extension is the core component of our browser automation platform. It allows you to record and automate browser actions without any coding experience.

Visual Recorder

Record your browser actions with our visual recorder. Simply click, type, and navigate as you normally would, and Selenix will capture every action.

Quick Access

Access your automations directly from the Chrome toolbar. Run, edit, or create new automations with just a few clicks.

Secure Authentication

Your login credentials are securely stored and encrypted. Selenix does not have access to your passwords or sensitive information.

Cloud Sync

Your automations are synced to the cloud, allowing you to access them from any device with the Selenix extension installed.

Extension Permissions

The Selenix Chrome extension requires certain permissions to function properly. Here's what each permission is used for:

Access browser tabs and navigation

This permission allows Selenix to record and automate actions across different tabs and websites.

Read and modify website content

This is necessary for automating form filling, clicking buttons, and extracting data from websites.

Download files

For automations that involve downloading files from websites.

Connect to Selenix servers

This allows the extension to sync your automations to your Selenix account and access your stored automations.

System Requirements

Selenix is designed to work on modern web browsers and operating systems. Here are the minimum requirements:

Browser

  • Google Chrome 88 or higher
  • Microsoft Edge 88 or higher (Chromium-based)
  • Brave Browser (Latest version)
  • Opera (Latest version)

Computer Requirements

  • 2 GB RAM minimum (4 GB recommended)
  • 1 GHz processor or faster
  • 100 MB of available disk space
  • Internet connection

Operating System

  • Windows 10 or higher
  • macOS 10.14 (Mojave) or higher
  • Ubuntu 18.04 or higher
  • Other Linux distributions (may vary)

Note

While Selenix may work on other browsers or older versions of Chrome, we officially support and test on the platforms listed above. For the best experience, we recommend using the latest version of Google Chrome.

Troubleshooting

Experiencing issues with the installation? Here are some common problems and their solutions:

Extension Won't Install

Problem: Chrome shows an error when trying to install the extension or the installation doesn't complete.

Solution:

  1. Make sure you're using a supported browser (Chrome, Edge, Brave, or Opera).
  2. Check your internet connection.
  3. Clear your browser cache and cookies.
  4. Temporarily disable other extensions that might interfere.
  5. Restart your browser and try again.

Extension Installed But Not Working

Problem: The extension is installed but doesn't appear in your toolbar or doesn't function properly.

Solution:

  1. Click the puzzle piece icon in Chrome to see if Selenix is listed in your extensions.
  2. If it's listed, click the pin icon to make it visible in your toolbar.
  3. Make sure the extension is enabled (toggle should be blue).
  4. Try restarting your browser.
  5. If the issue persists, try uninstalling and reinstalling the extension.

Can't Create an Account

Problem: Unable to create a new account after installing the extension.

Solution:

  1. Make sure you have a stable internet connection.
  2. Check if your email address is entered correctly.
  3. Try using a different email address.
  4. Ensure your password meets the minimum requirements (at least 8 characters, including a number and a special character).
  5. If you're still having issues, contact our support team at support@selenix.io.

Browser Compatibility Issues

Problem: The extension doesn't work correctly in your browser.

Solution:

  1. Update your browser to the latest version.
  2. If you're using a browser other than Chrome, try installing Selenix on Chrome instead.
  3. Check if your browser has any restrictions or security settings that might block extension functionality.
  4. Disable any ad blockers or privacy extensions that might interfere with Selenix.
