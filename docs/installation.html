Installing Selenix

Selenix is available for Windows, macOS, and Linux. Follow the instructions below to install Selenix on your platform.

System Requirements:

Selenix requires Chrome or Edge browser (version 88 or later) to be installed on your system.

Windows Installation

  1. Download the installer - Get the latest version from our download page.
  2. Run the installer - Double-click the downloaded .exe file and follow the installation wizard.
  3. Launch Selenix - Once installation is complete, you can launch Selenix from your Start menu or desktop shortcut.
Note for Windows Users:

If you encounter a Windows Defender SmartScreen warning, click "More info" and then "Run anyway". This happens because our installer is not yet widely distributed.

Windows installation wizard

macOS Installation

  1. Download the DMG file - Get the latest version from our download page.
  2. Open the DMG file - Double-click the downloaded file to mount it.
  3. Drag to Applications - Drag the Selenix icon to your Applications folder.
  4. Launch Selenix - Open your Applications folder and double-click on Selenix.
Note for macOS Users:

If you see a message saying "Selenix can't be opened because it is from an unidentified developer", right-click the app and select "Open", then click "Open" in the dialog that appears.

Drag Selenix to Applications folder

Linux Installation

Using the AppImage

  1. Download the AppImage - Get the latest version from our download page.
  2. Make it executable - Right-click the downloaded file and select "Properties", then go to "Permissions" and check "Allow executing file as program". Alternatively, use chmod +x Selenix-x.x.x.AppImage in the terminal.
  3. Run the AppImage - Double-click the AppImage file to launch Selenix.

Using the Debian package (.deb)

For Ubuntu, Debian, and other Debian-based distributions:

sudo apt update
sudo apt install ./selenix_x.x.x_amd64.deb

Using the RPM package (.rpm)

For Fedora, CentOS, and other RPM-based distributions:

sudo dnf install ./selenix-x.x.x.x86_64.rpm
Tip for Linux Users:

Selenix uses the Chrome browser under the hood. If you don't have Chrome installed, Selenix will prompt you to install it during the first launch.

Browser Extension

In addition to the desktop application, you'll need to install the Selenix browser extension to enable recording and automation:

Chrome Extension

  1. Open the Chrome Web Store - Go to the Selenix Chrome extension page.
  2. Add to Chrome - Click "Add to Chrome" and confirm the installation.
  3. Pin the extension - Click the extensions icon in the toolbar, find Selenix, and click the pin icon to keep it easily accessible.

Edge Extension

  1. Open the Edge Add-ons - Go to the Selenix Edge extension page.
  2. Add to Edge - Click "Get" and confirm the installation.
  3. Pin the extension - Click the extensions icon in the toolbar, find Selenix, and click the pin icon.
Selenix browser extension

Verifying Installation

To verify that Selenix is installed correctly:

  1. Launch the Selenix desktop app from your applications menu or shortcut.
  2. Check for the browser extension - Ensure the Selenix icon appears in your browser toolbar.
  3. Connect the extension - Click the extension icon and ensure it connects to the desktop app.
Success!

If you see "Connected to Selenix" in the extension popup, you're ready to start automating!

Configuration

Selenix works out of the box with default settings, but you may want to configure these options:

Application Settings

  • Default browser - Choose which browser Selenix should use for automation.
  • Data storage location - Change where Selenix stores workflows and data.
  • Proxy settings - Configure proxies for web scraping or accessing restricted websites.
  • Performance - Adjust speed and resource usage for automation runs.

Extension Settings

  • Recording mode - Choose between simple and advanced recording modes.
  • Selector preferences - Configure how Selenix identifies elements on web pages.
  • Permissions - Manage which websites Selenix can access.

Access settings through the gear icon in the main Selenix application or the extension options.

Troubleshooting

Common Installation Issues

Extension Not Connecting

If the browser extension can't connect to the desktop app, make sure the app is running. Restart both the browser and the app if needed.

Permission Issues

  • Windows: Try running the installer as Administrator.
  • macOS: Check System Preferences > Security & Privacy for blocked applications.
  • Linux: Ensure you have proper permissions for the installation directory.

Browser Compatibility

Selenix works best with:

  • Google Chrome (version 88+)
  • Microsoft Edge (version 88+)

Other Chromium-based browsers may work but are not officially supported.
