What is Selenix?

Selenix is a visual browser automation tool that allows you to:

  • Record and replay user interactions with websites
  • Extract data from web pages automatically
  • Schedule automations to run at specific times
  • Create complex workflows with conditional logic and loops
  • Export data to various formats (CSV, Excel, JSON)
The Selenix interface showing a workflow builder
