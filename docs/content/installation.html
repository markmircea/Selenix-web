Installing Selenix AI

Selenix is available for Windows, macOS, and Linux with built-in AI assistant capabilities. Follow the instructions below to install Selenix and unlock the power of AI-enhanced browser automation.

System Requirements:

Selenix requires Chrome or Edge browser (version 88 or later). For AI features, you'll need an OpenRouter API key (optional but recommended).

Windows Installation

  1. Download Selenix + License - Get the complete package from our download page (includes 1-year license automatically).
  2. Run the installer - Double-click the downloaded .exe file and follow the installation wizard.
  3. Launch Selenix - Once installation is complete, launch Selenix from your Start menu or desktop shortcut.
  4. License activation - Your license will be automatically activated during first launch.
Note for Windows Users:

If you encounter a Windows Defender SmartScreen warning, click "More info" and then "Run anyway". This happens because our installer is not yet widely distributed.

Windows installation wizard with AI assistant setup

macOS Installation

  1. Download the DMG file - Get the latest version with AI features from our download page.
  2. Open the DMG file - Double-click the downloaded file to mount it.
  3. Drag to Applications - Drag the Selenix icon to your Applications folder.
  4. Launch Selenix - Open your Applications folder and double-click on Selenix.
  5. License activation - Your included 1-year license will be automatically activated.
Note for macOS Users:

If you see a message saying "Selenix can't be opened because it is from an unidentified developer", right-click the app and select "Open", then click "Open" in the dialog that appears.

Drag Selenix to Applications folder

Linux Installation

Using the AppImage (Recommended)

  1. Download the AppImage - Get the latest version with AI features from our download page.
  2. Make it executable - Right-click the downloaded file and select "Properties", then go to "Permissions" and check "Allow executing file as program". Alternatively, use chmod +x Selenix-x.x.x.AppImage in the terminal.
  3. Run the AppImage - Double-click the AppImage file to launch Selenix.
  4. License activation - Your included license will be automatically activated on first run.

Using the Debian package (.deb)

For Ubuntu, Debian, and other Debian-based distributions:

sudo apt update
sudo apt install ./selenix_x.x.x_amd64.deb

Using the RPM package (.rpm)

For Fedora, CentOS, and other RPM-based distributions:

sudo dnf install ./selenix-x.x.x.x86_64.rpm
Tip for Linux Users:

Selenix uses the Chrome browser for automation. If you don't have Chrome installed, Selenix will prompt you to install it during the first launch.

AI Assistant Setup

To unlock the full power of Selenix's AI features, you'll need to configure the AI assistant:

Getting an OpenRouter API Key

  1. Visit OpenRouter - Go to OpenRouter.ai
  2. Create an account - Sign up for a free account
  3. Generate API key - Navigate to your dashboard and create a new API key
  4. Add credits - Add a small amount of credits to your account (typically $5-10 lasts months)

Configuring AI in Selenix

  1. Open Settings - In Selenix, go to Settings â†’ AI Configuration
  2. Enter API Key - Paste your OpenRouter API key
  3. Select Model - Choose from available models (Claude 3.5 Sonnet recommended)
  4. Test Connection - Click "Test AI Connection" to verify setup
AI Features Unlocked!

With AI configured, you can now use natural language commands, get intelligent automation suggestions, and leverage advanced debugging assistance.

Verifying Installation

To verify that Selenix is installed correctly with all features:

  1. Launch the Selenix desktop app from your applications menu or shortcut.
  2. Check license status - Verify that your 1-year license is active in the status bar.
  3. Test AI assistant - Open the AI Assistant panel and ask: "Can you help me create an automation?"
  4. Create a test project - Click "New Project" to ensure basic functionality works.
Installation Complete!

If the AI assistant responds and you can create projects, Selenix is ready for professional automation work!

Troubleshooting

Common Installation Issues

License Activation Failed

Solution: Check your internet connection and firewall settings. The license requires a one-time online activation.

AI Assistant Not Responding

Solution: Verify your OpenRouter API key is correct and has sufficient credits. Check the AI configuration in Settings.

Browser Not Detected

Solution: Install Chrome or Edge (version 88+). Restart Selenix after browser installation.

Security Software Blocking

Solution: Add Selenix to your antivirus whitelist. Some security software may block automation tools.

Permission Issues

  • Windows: Try running the installer as Administrator. Check Windows Defender exclusions.
  • macOS: Check System Preferences > Security & Privacy > Privacy > Automation for Selenix permissions.
  • Linux: Ensure you have proper permissions for the installation directory and browser access.

Performance Optimization

Memory Usage

For large-scale scraping, increase the memory allocation in Settings > Performance.

Execution Speed

Adjust timing settings based on target websites. Slower sites need longer timeouts.

Data Storage

Regularly clean up old exports and snapshots to maintain optimal performance.
