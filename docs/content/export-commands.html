Import & Export Commands

Selenix provides comprehensive data import and export capabilities, supporting multiple formats and automated file operations. These commands are essential for data workflows and integration with external systems.

exportToCSV

Professional

Exports a variable containing an array of objects to a CSV file. Perfect for spreadsheet analysis and data sharing.

Syntax

exportToCSV | variableName | filePath

Parameters

Target: Variable name containing array data to export
Value: File path where CSV will be saved (absolute or relative)

Examples

scrapeStructured | {"_root": "css=.product", "title": "css=.title", "price": "css=.price"} | products exportToCSV | products | ./exports/products.csv exportToCSV | userList | C:/data/users-${timestamp}.csv echo | Data exported successfully
Scrape product data and export to CSV with timestamp

CSV Features

  • Automatically creates headers from object keys
  • Handles nested objects by flattening them
  • Supports variable interpolation in file paths
  • Creates directories if they don't exist

exportToJSON

Professional

Exports a variable to a JSON file with proper formatting. Preserves data structure and types.

Syntax

exportToJSON | variableName | filePath

Examples

combineVariables | products,users,settings | completeData exportToJSON | completeData | ./exports/complete-data.json exportToJSON | scrapedData | C:/backups/data-${date}.json
Combine multiple datasets and export to JSON

JSON Benefits

  • Preserves complex data structures and arrays
  • Maintains data types (strings, numbers, booleans)
  • Human-readable with proper indentation
  • Can be imported back into Selenix

importFromJSON

Professional

Imports data from a JSON file and stores it in a variable. Variable type will be object or array depending on file contents.

Syntax

importFromJSON | filePath | variableName

Examples

importFromJSON | ./data/config.json | appConfig importFromJSON | C:/data/previous-scrape.json | existingData combineVariables | existingData,newData | mergedData echo | Loaded ${existingData.length} existing records
Import configuration and merge with new data

importFromCSV

Professional

Imports data from a CSV file and stores it as an array of objects. Each row becomes an object with column headers as keys.

Syntax

importFromCSV | filePath | variableName

Examples

importFromCSV | ./data/customers.csv | customerList forEach | customerList | customer echo | Processing customer: ${customer.name} end
Import customer data and process each record

CSV Import Features

  • Automatically detects headers from first row
  • Handles quoted fields and special characters
  • Converts numeric strings to numbers when appropriate
  • Skips empty rows automatically

downloadFiles

Advanced

Downloads files from an array of URLs automatically. Handles multiple files concurrently with progress tracking.

Syntax

downloadFiles | urlArrayVariable | downloadPath

Parameters

Target: Variable containing array of URLs to download
Value: Directory path where files will be saved

Examples

scrapeCollection | css=.download-link@href | fileUrls downloadFiles | fileUrls | ./downloads/ scrapeCollection | css=img@src | imageUrls downloadFiles | imageUrls | C:/images/
Scrape download links and images, then download them

Download Features

  • Concurrent downloads for faster processing
  • Automatic file naming based on URLs
  • Progress tracking and error handling
  • Supports various file types (images, documents, etc.)

httpRequest

Advanced

Sends HTTP requests and stores the response. Perfect for API integration and data fetching.

Syntax

httpRequest | requestConfig | variableName

Parameters

Target: JSON configuration for the HTTP request
Value: Variable name to store the response (optional)

Examples

httpRequest | {"url": "https://api.example.com/data", "method": "GET", "headers": {"Authorization": "Bearer ${token}"}} | apiResponse httpRequest | {"url": "https://webhook.site/unique-id", "method": "POST", "body": {"data": "${scrapedData}"}} | postResult echo | API returned: ${apiResponse.status}
Make GET and POST requests with authentication

curlRequest

Advanced

Executes cURL-style HTTP requests natively. Perfect for complex API calls and integrations.

Syntax

curlRequest | curlCommand | variableName

Examples

curlRequest | curl -X GET "https://api.github.com/users/octocat" -H "Accept: application/json" | githubUser curlRequest | curl -X POST "https://httpbin.org/post" -d "data=${scrapedData}" | postResponse
Execute cURL commands for API interactions
