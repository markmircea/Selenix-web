Data Management Commands

Selenix provides sophisticated data management capabilities for storing, transforming, and processing variables. These commands enable complex data workflows and intelligent automation.

store

Basic

Save a target string as a variable for easy re-use. Creates a string-type variable.

Syntax

store | text | variableName

Examples

store | Hello World | greeting store | https://example.com | baseUrl store | ${firstName} ${lastName} | fullName echo | ${greeting}
Store simple values and combine variables

storeJson

Advanced

Save JSON as an object in a variable. Creates an object-type variable that can be accessed with dot notation.

Syntax

storeJson | jsonString | variableName

Examples

storeJson | {"name": "John", "age": 30, "city": "New York"} | user storeJson | [{"id": 1, "title": "Task 1"}, {"id": 2, "title": "Task 2"}] | tasks echo | User name: ${user.name} echo | First task: ${tasks[0].title}
Store structured data and access properties

transformVariable

Professional

Transforms a variable using a JavaScript function. Powerful for data cleaning, formatting, and processing.

Syntax

transformVariable | variableName | javascriptFunction

Examples

transformVariable | prices | data => data.map(price => parseFloat(price.replace('$', ''))) transformVariable | usernames | data => data.filter(name => name.length > 3) transformVariable | text | data => data.toUpperCase().trim() transformVariable | dates | data => data.map(d => new Date(d).toISOString())
Clean prices, filter data, format text, and convert dates

JavaScript Functions

  • Use arrow functions: data => data.map(...)
  • Access array methods: filter, map, reduce
  • String methods: replace, trim, split
  • Number parsing: parseInt, parseFloat

combineVariables

Professional

Combines multiple variables into one with intelligent type handling. Perfect for preparing data for export or creating comprehensive datasets.

Syntax

combineVariables | variablesList | resultVariableName

Parameters

Target: Comma-separated list of variables OR JSON configuration
Value: Variable name to store the combined result

Examples

combineVariables | titles,prices,descriptions | productData combineVariables | {"variables":["users","products"],"mode":"object"} | combinedData combineVariables | {"variables":["firstName","lastName"],"mode":"string","options":{"separator":" "}} | fullName
Simple combination, object mode, and string concatenation

Combination Modes

  • Auto-detection: Intelligently determines the best combination method
  • Array mode: Concatenates arrays, converts objects to arrays
  • Object mode: Merges objects, handles key conflicts
  • String mode: Concatenates with custom separators

inspectVariable

Utility

Display detailed information about a variable including its type, structure, and contents. Essential for debugging data workflows.

Syntax

inspectVariable | variableName

Examples

scrapeCollection | css=.product | products inspectVariable | products transformVariable | products | data => data.slice(0, 10) inspectVariable | products
Scrape data, inspect it, transform it, then inspect again

Inspection Details

  • Variable type (string, array, object, number, boolean)
  • Array length and sample elements
  • Object properties and nested structure
  • Memory usage for large datasets

executeScript

Advanced

Executes JavaScript in the browser context. Can access page elements and return values to variables.

Syntax

executeScript | javascriptCode | variableName

Examples

executeScript | return document.title | pageTitle executeScript | return document.querySelectorAll('.item').length | itemCount executeScript | window.scrollTo(0, document.body.scrollHeight) executeScript | return localStorage.getItem('userPrefs') | userPrefs
Get page info, count elements, scroll, and access localStorage

executeAsyncScript

Advanced

Executes asynchronous JavaScript that returns a Promise. Perfect for API calls and async operations.

Syntax

executeAsyncScript | asyncJavaScriptCode | variableName

Examples

executeAsyncScript | const response = await fetch('/api/data'); return response.json(); | apiData executeAsyncScript | return new Promise(resolve => setTimeout(() => resolve('done'), 2000)) | result
Make API calls and handle async operations

forEach

Control Flow

Create a loop that executes commands for each item in an array variable. Must be terminated with an "end" command.

Syntax

forEach | arrayVariableName | iteratorVariableName

Examples

scrapeCollection | css=.product-url | productUrls forEach | productUrls | currentUrl open | ${currentUrl} storeText | css=.product-title | title echo | Processing: ${title} end
Iterate through URLs and process each product page
