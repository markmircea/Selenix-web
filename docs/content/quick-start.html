Quick Start Guide

Welcome to Selenix! This guide will help you create your first AI-powered automation in just 5 minutes, using our intelligent assistant to guide you through the process.

Step 1: Launch Selenix

  1. Open the Selenix desktop application
  2. Click Create Project to start
  3. Optionally, add your OpenRouter API key in Config to unlock AI features
  4. Click on the Tests tab
  5. Click the + on the project drawer to create a new test
  6. Click the + in the commands area to add a new command
AI Assistant Available

Look for the AI Assistant panel on the right side of the interface. It can help you throughout the automation process!

Step 2: Create with AI Assistance

Let's create an intelligent web scraping automation:

  1. Open the built-in browser and navigate to any website (e.g., example.com)
  2. Ask the AI Assistant: "Help me scrape all the product titles from this page"
  3. The AI will analyze the page and suggest optimal commands
  4. Import the suggested commands, targets and values given by the AI and run the test
  5. *It is recommended to start with one element at a time, for example, usernames first, then after it's working continue building on the existing workflow*
AI Magic!

The AI Assistant can see your browser by using screenshots, understand page structure by accessing the underlying HTML, and access the current workspace (user projects/suites/commands and logs).

Step 3: Advanced Data Extraction

Try these powerful scraping commands:

Scrape Collection

Extract multiple similar items:

scrapeCollection | css=.product-title | productTitles

This extracts all product titles into an array variable

Here is what the return format would look like for 2 articles (Array of strings)

"We now have 25x brand new nodes in n8n",
"Advice Needed : Linking Dynamic GPT SQL to API"
]

Export Data

Save your scraped data:

exportToCSV | productData | C:/exports/products.csv

This exports your data to a CSV file

Step 4: Run and Schedule

  1. Click the Play button to test your automation
  2. Watch as Selenix intelligently performs your actions
  3. Check the exported data in your specified location
  4. In the tests drawer, select Schedule Test (Clock Icon)
  5. Set up automated runs, Once / Recurring (hourly, daily, weekly, monthly)

One-time Schedule

Run once at a specific date and time

Recurring Schedule

Run automatically on a recurring pattern

Step 5: Browser State Management

Save and restore browser states for consistent automation:

  1. Add command: createSnapshot | myLoginState
  2. This saves cookies, localStorage, and session data in the variable myLoginState
  3. To use the session in another test you can export
    Export to JSON | myLoginState | C:/projects/snapshot.json
  4. Later in another test, use:
    Import from JSON | myLoginState | C:/projects/snapshot.json
    restoreSnapshot | myLoginState
  5. Your browser will return to the exact saved state

Pro Tips for AI-Powered Automation

Talk to Your AI

Describe what you want in natural language: "I need to scrape all product prices and export to CSV"

Use Screenshots

The AI can analyze browser screenshots to help identify elements and troubleshoot issues

Schedule Smart

Set up recurring schedules for data collection. Perfect for monitoring competitors or tracking changes

Save States

Create browser snapshots after logging in to avoid re-authentication in scheduled runs

Essential Commands to Try

Data Extraction

  • scrapeCollection - Extract arrays of similar elements
  • scrapeStructured - Extract complex data structures
  • storeText - Store text from single elements

Data Processing

  • combineVariables - Merge multiple data sources
  • transformVariable - Apply JavaScript transformations
  • sendToAI - Process data with AI instructions

Export & Import

  • exportToCSV - Export data to CSV files
  • exportToJSON - Export data to JSON files
  • importFromJSON - Import data from files

State Management

  • createSnapshot - Save browser state
  • restoreSnapshot - Restore browser state
  • monitorElement - Watch for changes
